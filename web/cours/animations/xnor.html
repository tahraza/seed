XNOR(A, B) = NOT(XOR(A, B))

La porte XNOR (equivalence) est une porte XOR suivie d'un inverseur. Elle retourne 1 quand les deux entrees sont egales (toutes deux 0 ou toutes deux 1).

Verification:
XOR(0, 0) = 0 → NOT(0) = 1 (egaux) ✓
XOR(0, 1) = 1 → NOT(1) = 0 (differents) ✓
XOR(1, 0) = 1 → NOT(1) = 0 (differents) ✓
XOR(1, 1) = 0 → NOT(0) = 1 (egaux) ✓
