Voir la Construction

▼
A = AND(IN, NOT(SEL))
B = AND(IN, SEL)

Le demultiplexeur est l'inverse du MUX: il route le signal d'entree vers une des sorties. Chaque sortie est activee par une combinaison du signal IN et du selecteur.

Fonctionnement:
Si SEL=0: A = IN (signal passe vers A), B = 0
Si SEL=1: A = 0, B = IN (signal passe vers B)
